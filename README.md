# pymin_reqs

## Install With Pip
> `python3 -m pip install git+https://github.com/parnell/pymin-reqs`

## Usage
```
usage: pymin_reqs [-h] [-d DIRECTORY] [-o] [-v] [--outfile OUTFILE]

optional arguments:
  -h, --help            show this help message and exit
  -d DIRECTORY, --directory DIRECTORY
  -o, --overwrite
  -v, --verbose
  --outfile OUTFILE
```
### Example Usage
> `pymin_reqs`
